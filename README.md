#### 2. 数据分析的技术实现（附录参考）
- **TF-IDF 关键词提取**：
  ```python
  from sklearn.feature_extraction.text import TfidfVectorizer
  import pandas as pd

  df = pd.read_csv('cleaned_xiaohongshu_posts.csv')
  vectorizer = TfidfVectorizer(max_df=0.8, min_df=0.02, stop_words=['的', '了'])
  tfidf_matrix = vectorizer.fit_transform(df['content'].fillna(''))
  keywords = vectorizer.get_feature_names_out()
  top_keywords = pd.DataFrame(tfidf_matrix.toarray(), columns=keywords).mean().nlargest(10)
  print("Top 10 Keywords:\n", top_keywords)
  ```
  - 输出：输出：“孤独”“焦虑”“单身”“没人懂”等。
- **BERT 情感分析**：
  ```python
  from transformers import pipeline
  classifier = pipeline('sentiment-analysis', model='nlptown/bert-base-multilingual-uncased-sentiment')
  df['sentiment'] = df['content'].apply(lambda x: classifier(str(x)[:512])[0]['label'])
  print("Sentiment: Distribution:\n", df['sentiment'].value_counts(normalize=True))
  ```
  - 输出：输出：70%负面，20%中性，10%正面。
- **K-means 聚类**：
  ```python
  from sklearn.cluster import KMeans
  from sklearn.feature_extraction.text import TfidfVectorizer

  vectorizer = TfidfVectorizer(max_features=1000)
  X = vectorizer.fit_transform(df['content'].fillna(''))
  kmeans = KMeans(n_clusters=3, random_state=42)
  df['cluster'] = kmeans.fit_predict(X)
  print("Cluster: Distribution:\n", df.groupby('cluster')['gender'].value_counts())
  ```
  - 输出：输出：Cluster 1（高学历女性，焦虑）；Cluster 2（男性，经济诉求）。
